این پروژه برای ساخت نسخه‌ی ویندوز (EXE) آماده شد.

مراحل محلی برای ساخت EXE:
1) در پوشه‌ی پروژه که package.json قرار دارد:
   npm install

2) ساخت فایل‌های وب (vite build):
   npm run build

3) ساخت فایل نصبی ویندوز:
   npm run electron-build

توجه:
- بهتر است این کار را روی ویندوز انجام دهید. اگر روی لینوکس هستید و می‌خواهید برای ویندوز بسازید، نیاز به wine و تنظیمات اضافی است.
- اگر می‌خواهید حالت توسعه با ریلود ببینید:
   npm install
   npm run dev
   در ترمینال دیگری:
   npm run electron

فایل‌های اضافه شده: electron.js, preload.js
خروجی build در پوشه‌ی release/ قرار می‌گیرد.
